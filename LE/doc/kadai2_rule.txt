PROGRAM -> FUNCTION
FUNCTION -> TYPENAME IDENTIFIER LEFT_PARENTHESIS ARGUMENT_LIST RIGHT_PARENTHESIS BLOCK
ARGUMENT_LIST -> ARGUMENT_LIST COMMA ARGUMENT | ARGUMENT
ARGUMENT -> TYPENAME IDENTIFIER | VOID

WHILE_STATEMENT -> WHILE LEFT_PARENTHESIS EXPRESSION RIGHT_PARENTHESIS BLOCK

IF_STATEMENT -> IF LEFT_PARENTHESIS EXPRESSION RIGHT_PARENTHESIS BLOCK

BLOCK -> STATEMENT | LEFT_BRACE STATEMENT_LIST RIGHT_BRACE
STATEMENT_LIST -> STATEMENT | STATEMENT_LIST STATEMENT
STAREMENT -> EXPRESSION SEMICOLON | TYPENAME VARIABLE_LIST SEMICOLON
						 | IF_STATEMENT | WHILE_STATEMENT
TYPENAME -> INT | DOUBLE
VARIABLE_LIST -> IDENTIFIER | VARIABLE_LIST COMMA IDENTIFIER
EXPRESSION -> TERM | LEFT_PARENTHESIS EXPRESSION RIGHT_PARENTHESIS
							| EXPRESSION OPERATOR TERM
OPERATOR -> ADD | SUB | MUL | DIV | EQUAL | GREATER | LESS
TERM -> NUMBER_INT | NUMBER_HEX | NUMBER_REAL | IDENTIFIER
