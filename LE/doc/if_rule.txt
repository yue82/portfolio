IF_STATEMENT -> : IF LEFT_PARENTHESIS EXPRESSION RIGHT_PARENTHESIS BLOCK

BLOCK -> STATEMENT | LEFT_BRACE STATEMENT_LIST RIGHT_BRACE
STATEMENT_LIST -> STATEMENT | STATEMENT_LIST STATEMENT
STATEMENT -> EXPRESSION SEMICOLON | TYPENAME VARIABLE_LIST SEMICOLON
VARIABLE_LIST -> IDENTIFIER | VARIABLE_LIST COMMA IDENTIFIER
EXPRESSION -> TERM | LEFT_PARENTHESIS EXPRESSION RIGHT_PARENTHESIS | EXPRESSION OPERATOR TERM
TERM -> INTEGER | REAL | HEXADECIMAL | IDENTIFIER
